AI API KEY FOR GRADERS
====================================

QUICK SETUP:

1. Open .env file and replace 'your_api_key_here' with your actual API key:
   API_KEY=your-actual-api-key-here

2. Run the app:
   setup_and_run.bat


Note: The app works without an API key (geometric classification only).
With the API key, you get enhanced AI-powered surface descriptions.
